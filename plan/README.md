# 学习计划目录

本目录用于存放基于 POE 中 Chat 机器人创建的复习计划及周报告。

## 目录结构
- `weekly/` - 存放每周的具体计划和复盘文档
- `templates/` - 计划模板
- `reviews/` - 周/月度复盘总结

## 使用指南
1. 每周计划存放在 `weekly` 目录下，以 `week-X.md` 格式命名
2. 每周末进行复盘，总结存放在 `reviews` 目录
3. 根据复盘结果，适时调整下周计划
